Installation
############

There are two options available for installing the Tiny Webcasts command-line tool ``twc``:
  
  #. Build and use the docker image provided.

  #. Install "Tiny Webcasts" directly on your operating sytem.


**Windows users** (and macOS users) are highly recommended to use the docker image.


Installing the Tiny Webcasts command-line tool is fairly simple and consists of only two steps:


  #. Download the project from `GitHub <https://github.com/tillbiskup/tiny-webcasts>`_.

  #. Install the ``twc`` command-line tool by running the appropriate installer.


For convenience, we provide you with detailed installation instructions for each of the three operating systems supported.
